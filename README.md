# 🚌 Bus Ticketing System

## 📌 Overview
This is a web-based **Bus Ticketing System** developed using Flask.  
It allows users to book bus tickets, manage bookings, and simulate payment processing.

---

## 🚀 Features
- 🔐 User Authentication (Login/Register)
- 🎟️ Ticket Booking System
- 🧾 Ticket Management
- 💳 Payment Module (Basic / Simulated)
- 🛠️ Admin Panel

---

## 🛠️ Technologies Used
- Python (Flask Framework)
- SQLite Database
- HTML, CSS (Jinja2 Templates)
- JavaScript (basic)
